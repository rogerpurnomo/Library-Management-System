# Library-Management-System

A simple Library Management System built using SQL, demonstrating core database concepts including DDL (Data Definition Language) and DML (Data Manipulation Language) operations. This project focuses on managing library data efficiently through structured queries and relational design.

This system simulates a real-world library database where users can:
- Manage books, members, and borrowing records
- Perform CRUD operations (Create, Read, Update, Delete)
- Maintain data integrity using relational database design
The project is designed to strengthen understanding of database fundamentals and SQL query implementation.

Learning Objectives
- Understand relational database design
- Practice SQL CRUD operations
- Apply DDL and DML commands
- Build structured and normalized tables

Note = Entity Relationship Diagram is provided in docx file, feel free to explore!
